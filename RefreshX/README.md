# RefreshX - Wellness App for Digital Workers

RefreshX is an iOS app designed to help desk workers prevent health issues by taking strategic breaks for eye, back, and wrist exercises.

## Features

- **Customizable Break Scheduling**: Set up work hours and configure breaks based on your schedule
- **Targeted Exercises**: Specialized exercises for eyes, back, and wrists
- **Progress Tracking**: Monitor your wellness journey with detailed statistics
- **Educational Content**: Articles on workplace wellness and proper ergonomics
- **Personalized Routines**: Create custom exercise routines for your needs

## App Sections

1. **Home**: Dashboard showing next break, activity summary, and progress overview
2. **Exercises**: Library of all available exercises categorized by focus area
3. **Explore**: Educational articles and wellness content
4. **Progress**: Detailed statistics and activity history
5. **Profile**: User settings, break schedule configuration, and app preferences

## Technical Information

- **Platform**: iOS 15.0+
- **Framework**: SwiftUI
- **Backend**: Supabase
- **Architecture**: MVVM with Combine

## Getting Started

1. Clone the repository
2. Open the project in Xcode 13.0+
3. Install dependencies (if any)
4. Run the project on a simulator or device

## Data Models

RefreshX uses the following key data models:

- **User**: Stores user account information and preferences
- **Exercise**: Represents a single exercise with instructions
- **Routine**: Collection of exercises for a user
- **BreakSession**: Records of break activities
- **Article**: Educational content
- **Progress**: Statistics about user activity
